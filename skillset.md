**Management**

- [GitLab](https://about.gitlab.com/) for our code repository and [CI/CD](https://docs.gitlab.com/ee/ci/)
- [Trunk Based Development](https://trunkbaseddevelopment.com/) with Git
- [Jira](https://www.atlassian.com/software/jira) for product/project management (Scrum)
- [SonarQube](https://www.sonarqube.org/) for secure code analysis
- [LastPass](https://www.lastpass.com/) for password management

**Backend**

[Node](https://nodejs.org/en/about/)/[Express](https://expressjs.com/): It's important to understand the [event loop](https://nodejs.org/en/docs/guides/event-loop-timers-and-nexttick/) and [blocking/non-blocking](https://nodejs.org/en/docs/guides/blocking-vs-non-blocking/) in Node

[Mocha](https://mochajs.org/) and [Chai](https://www.chaijs.com/) for testing

[mysql](https://github.com/mysqljs/mysql) for our MySQL JS client

[Postman](https://www.postman.com/) for manual API testing and documentation [MySQL Workbench](https://www.mysql.com/products/workbench/) for direct database work

[Twilio](https://www.twilio.com/docs/sms) for SMS

**Frontend**

[React](https://reactjs.org/docs/getting-started.html): especially main concepts and hooks (covered in the docs)

Chrome plugins: [Locale Switcher](https://chrome.google.com/webstore/detail/locale-switcher/kngfjpghaokedippaapkfihdlmmlafcc?hl=en-US), [Axe Dev Tools](https://chrome.google.com/webstore/detail/axe-devtools-web-accessib/lhdoppojpmngadmnindnejefpokejbdd?hl=en-US)

[Material UI](https://material-ui.com/) for frontend UI

[Internationalization](https://react.i18next.com/)

Main AWS Services to familiarize yourself with: Cognito, Lambda, Elastic Beanstalk, RDS (MySQL 8.0), EC2 (Linux)
